# Git básico: Commit, Push y Pull

## 1. Commit
Guarda los cambios realizados en el repositorio local.

git add .
git commit -m "Descripción de los cambios"

## 2. Push
Envía los commits del repositorio local al repositorio remoto.

git push origin nombre-de-rama

## 3. Pull
Descarga y actualiza tu repositorio local con los cambios del repositorio remoto.

git pull origin nombre-de-rama
